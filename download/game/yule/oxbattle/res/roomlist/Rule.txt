基本规则
一、 牌型大小：
本牌型依次大小为：大王、小王、K、Q、J、10、9、8、7、6、5、4、3、2 、A
 
二、 出牌顺序：
1、  发牌顺序是按随机抽出一张牌，按牌的点数计算先分给哪一家，比如抽出的亮牌是A那么就先分给庄家，如是2就先分闲家1；如是3就先分给闲家2；如是4就先分给闲家3如是5就先分闲家4；如是六就先分给庄家，以此类推。
2、  发牌后的第4张先不让人看到，增加动画功能，像两张一样。先由闲家1开始亮牌，其次是闲家2闲家3闲家4最后庄家亮牌。
 
三、 牌型
.1、特殊牌型
大王、小王可以当任点数。J、Q、K都是当10点，然后A是当1点，其他的牌型当自身的点数。
4.2、牌型组合
牌局开始每个人手中都有五张牌，然后玩家需要将手中任意三张牌凑成10点或20点或30点都可以，这样是被称之为牛。接下来在将其余的两张的点数相加得出几点。去掉十位数，只留个位数来进行比较，如果接下来两张的相加点数也正好是整数的话，那就是最大的牌型：“牛牛”。
 
四、 牌型大小比较
当庄家与闲家同时出现相同点数时，系统自动将两家手中牌的最大那一张进行比较，谁大就由谁获得胜利。如果出现牌也相同大的话，就按花色来进行比较，花色的比较与梭哈的花色比较类同。花色：黑>红>梅>方
 
五、 获胜条件
游戏为百人玩，一方为庄家，其它为闲家，开始后由闲家向桌面掷筹码，掷完筹码后系统开始发牌，最后比较大小，闲家的牌点数大于庄家则算胜利。先决条件是：牛牛大于任何牌点数，牛9（前三张的组合成整数称之为“牛”）大于牛8。有牛的牌大于无牛的牌，玩家手中任意的三张牌组合不成一个整数，则被称之为无牛牌，无牛牌的比较就按牌中最大的那张进行比较。
注意：启牌的牌必须要三张牌组合成牛才可以，不可以由两张或者四张组合而成，如果三张组合不成牛的话，那这牌就属于无牛牌。
 
六、 计分
炸弹—10倍
金牛—10倍
银牛—10倍
牛牛——10倍与压注筹码
牛9——9倍与压注筹码
牛8——8倍与压注筹码
牛7——7倍与压注筹码
牛6——6倍与压注筹码
牛5——5倍与压注筹码
牛4——4倍与压注筹码
牛3——3倍与压注筹码
牛2——2倍与压注筹码
庄家与闲家之间的记分方式是固定。庄家赢钱的数量与闲家输钱的数量是成正比，比如：闲家押筹码为100两，如果此时庄家摸到牛牛此种大牌，闲家刚才所压的100两需要翻十倍给庄家。所以两者之间存在的输赢筹码数量是一个固定量。
百人牛牛要有金牛、银牛、炸弹，金牛就是五张牌全部是10以上的牌，10以上的牌有J、Q、K，金牛比牛牛大，其中金牛为10倍。银牛就是五张牌中4张牌是10以上的牌（包括10在内），银牛为10倍。百人牛牛中应该还要增加炸弹，炸弹为最大的牌，也是10倍。炸弹的牌型：如：AAAAX、2222X、3333X 、4444X、一直到KKKKX、。其的X是任何牌都可以。
注：在牛牛里面,出现有大王的是最大的牛牛,有小王的是第2小的牛牛，牛牛是最小的。（牛牛按照这个大王、小王、K、Q、J、10、9、8、7、6、5、4、3、2 、A）上面的意思就是出现有大王的牛牛,是牛牛中最大的牌,出现有小王的牛牛是牛牛中第二大的牌。牛牛比银牛小,银牛比金牛小,金牛比炸弹小
例如：两家都是银牛,其中一家有大王,也是银牛中的最大的银牛
 
七、 逃跑扣分
Ø  逃跑：
n   游戏进行中，
n   闲家未下注，直接退出逃跑，不扣分。
n   闲家下注了，未开牌，直接退还所下注金额。
n   下注了，开牌时间，逃跑，按正常结算。
Ø  庄家逃跑：
n   不管什么时候，都按正常结算。
 
八、 超级抢庄功能
超级抢庄是一种特殊的上庄手段，玩家消耗一定的游戏币，就可以直接超过其他的正常上庄者，快速成为庄家
Ø  抢庄规则
n   如果目前庄家是普通庄家，玩家在本局游戏中使用了超级抢庄，则直接排到上庄列表中的第一位置，等待本局游戏结束，下局游戏直接成为庄家
n   如果目前庄家已经是使用了超级抢庄的庄家，则使用了“超级抢庄”功能的玩家为上庄玩家的第一序列，等待超级庄家下庄后，玩家才能成为庄家
n   系统只允许一名玩家使用超级抢庄，如果多人同时点击，则判断第一个人成功，其余人失败。
n   当有人已使用超级抢庄，其余人无法使用该按钮（灰色）
n   没满足条件的人，按钮也是灰色
n   超级抢庄无法取消申请
Ø  超级抢庄配置条件：运营商可在服务器配置超端抢庄的使用条件
n   VIP等级（二选一）：玩家必须达到VIP X等级，才可以使用超级抢庄
n   携带游戏币超过设定值（二选一）：玩家身上必须携带超过X 游戏币，才可以使用超级抢庄
 
九、 座位功能（仅限手机）
 
座位功能是一种特殊的玩家显示功能，直接将玩家的头像显示在主界面中。在百人游戏，如百人牛牛和新百家乐中，房间会有很多的玩家，只能在玩家列表中找到，座位功能能让玩家将自己直接展示给所有玩家，是一种身份的象征。
目前该功能只在手机版百人牛牛中实现，PC版百人牛牛没有该功能。
 
Ø  座位条件配置：
n   上座条件（三选一）VIP等级：玩家必须达到VIP X等级，才可以使用超级抢庄。
n   上座条件（三选一）携带游戏币：玩家身上必须携带超过X 游戏币，才可以使用超级抢庄。
n   上座条件（三选一）消耗游戏币：玩家可以自由上座，但每次上座必须消耗 X游戏币。
n   自动离座条件：游戏币设定值：当玩家身上携带的游戏币少于X时，自动让该玩家离座。"
